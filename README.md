# Polaris

## Installazione

### Python Environment

Il progetto è stato sviluppato e testato con Python 3.13.9.

In Windows:
1. Aprire il terminale nella cartella di progetto e creare il python environment con `python -m venv venv`
2. Attivare l'environment con `.\venv\Scripts\activate`
3. Installare i pacchetti python necessari con `pip install -r requirements.txt`

### VSCode

In Windows:
1. Installare VSCode da https://code.visualstudio.com/
2. Aprire VSCode, andare su *Extensions* nel menù laterale e installare l'estensione *Jupyter* 

## Apertura

In Windows:
1. Aprire il terminale nella cartella di progetto
2. Digitare `code .`
3. Aprire il file *lab.ipynb* (importante aprirlo)
4. Con il focus su *lab.ipynb*, selezionare il kernel *venv*:
    4.1 Premere *Shift+Control+P*, 
    4.2 Selezionare *Notebook: Select Notebook Kernel*
    4.3 Premere su *venv*
